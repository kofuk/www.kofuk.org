Ubuntu (をはじめ多くの GNU/Linux ディストリビューション) では C 言語のマニュアルページは最初から標準で入っていますが、
C++ のマニュアルページは入っていません (C++ にはマニュアルページを見る文化がないのだろうか…)。
という訳で、自分でインストールする必要があります。

```console
$ sudo apt install libstdc++-8-doc
```

で、見るときは、

```console
$ man std::vector
```

みたいな感じで `std::` までつけないといけないみたいです。
