なんじゃこりゃ

```c
%:include <stdio.h>

int main(void) <%
    char str<::> = "Hello, world!";
    printf("%s\n", str);

    return 0;
%>
```

これが

```shell
$ gcc main.c
```

コンパイルすると

```shell
$ ./a.out
Hello, world
```

動いてしまう。

`<:`, `:>`, `<%`, `%>`, `%:`, `%:%:` の6つのトークンは，
`[`, `]`, `{`, `}`, `#`, `##` と同じように解釈されるらしい。

なんじゃそりゃ。
